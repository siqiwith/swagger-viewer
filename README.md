# swagger-viewer
 docker compose file for swagger


# Steps to use:
- clone the repo
```sh
git clone https://github.com/siqiwith/swagger-viewer.git
```
- cd `swagger-viewer`
- config `.env` file
- start services
```
docker-composer up
```
- view your swagger ui in browser
```
http://localhost
```
- input the yaml file path
```
http://localhost:3000/<API_NAME>.yaml
```
